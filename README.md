## Overview
This project is a **Hello World Django web application** built using **Python and Django** as a mini project for learning the fundamentals of web development.

The application demonstrates how to create a Django project with a single app, define a model, connect URLs to views, interact with a database, and render dynamic content on a web page.

The app displays a message stored in the database on the homepage ('Hello World!').

---

## Features
-  **Django Project Setup**  
  - Clean project and app structure  

-  **Database Integration**  
  - SQLite database (default Django database)  
  - Simple model for storing messages  

-  **Dynamic Web Page**  
  - Retrieves data from the database  
  - Renders content using Django templates  

-  **Admin Panel**  
  - Add and manage messages through Django Admin
